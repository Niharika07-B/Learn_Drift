# LearnDrift Frontend

AI-based Educational Analytics Platform - React Frontend

## 🚀 Quick Start

```bash
npm install
npm start
```

Open **http://localhost:3000**

## 📚 Documentation

- **START_HERE.md** - Quick start guide
- **THEME_FEATURES.md** - Theme system features

## 🎯 Features

✅ Beautiful dark/light theme toggle
✅ Student & Instructor dashboards
✅ Interactive charts and analytics
✅ Responsive design
✅ Mock login system
✅ Real-time data visualization

## 🌙 Theme System

- **Dark Mode**: Galaxy background with animated stars
- **Light Mode**: Professional gradient design
- **Toggle**: Click sun/moon icon in navbar
- **Persistent**: Theme preference saved automatically
